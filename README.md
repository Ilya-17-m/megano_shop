## Разработка:

### Требования:

#### Для установки и запуска проекта, необходим Python:3.12

### Установка зависимостей:
#### Для сбора контейнера выполните команду:
``docker build -t megano .``


#### Чтобы установить и запустить сразу все контейнеры, выполните команду:
``docker compose up --build``

### Запуск сервера:
#### Чтобы запустить сервер для разработки, выполните команду находясь в корне проекта:
``poetry run gunicorn backend.megano.megano.wsgi:application``


### Чтобы увидеть документацию проекта, выполните предыдущую команду в терминале и перейдите по ссылке:

#### Документация в формате swagger:
``http://127.0.0.1:8000/swagger``

#### Документация в формате redoc:
``http://127.0.0.1:8000/redoc``
